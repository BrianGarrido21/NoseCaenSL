◆ login()

login ( Request $request)

Procesa el inicio de sesión de un empleado.

Valida las credenciales ingresadas y, si son correctas, inicia sesión y redirige al dashboard del empleado. Si las credenciales son inválidas, retorna al formulario de login con errores.

Parameters
Request$requestDatos enviados desde el formulario de login.
Returns
RedirectResponse Redirección al dashboard o de vuelta al login con errores.

◆ logout()

logout ( Request $request)

Cierra la sesión del empleado autenticado.

Invalida la sesión actual y regenera el token de sesión para evitar ataques CSRF, redirigiendo nuevamente al login.

Parameters
Request$requestSolicitud actual para invalidar la sesión.
Returns
RedirectResponse Redirección al formulario de login.
